# Flexbox 布局

# 一、Flex Container

# 1. flex-direction

row

row-reverse

flex-direction: row 或 row-reverse
main-axis（主轴）：row
cross-axis（交叉轴）：column

column

column-reverse

flex-direction: column 或 column-reverse
main-axis（主轴）：column
cross-axis（交叉轴）：row

# 2. justify-content

主轴的排列方式

justify-content:
 	center: 居中
	flex-start: 主轴开始位置
	flex-end: 主轴结束位置
	space-evenly: 主轴方向所有 flex-item 间隙相同  |2A2B2C2D2|
	space-around: 主轴方向，flex-item 左右间隙相同  |1A2B2C2D1|
	space-between: 主轴方向，flex-item 头尾无间隙，其他元素之间间隙相同 |A2B2C2D|

# 3. align-items

交叉轴的排列方式

同上

# 4. flex-wrap: 会不会分行 默认：nowrap

flex-wrap: nowrap; // 不分行，内容挤压
flex-wrap: wrap; // 自动换行，不挤压

# 5. flex-flow: flex-direction + flex-wrap

flex-flow: row wrap;
// 等价于：行排列+自动换行
flex-direction: row;
flex-wrap: wrap;

# 6. align-content

align-content 在 flex-wrap 为 wrap 时生效，主要作用于多行 flex-item，行与行之间的对齐方式

与 align-item 有些类似，但又不同
align-items:属性适用于所有的flex容器，它是用来设置每个flex元素在交叉轴上的默认对齐方式。
align-items和align-content有相同的功能，不过不同点是它是用来让每一个单行的容器居中而不是让整个容器居中。
align-content属性只适用于多行的flex容器，并且当交叉轴上有多余空间使flex容器内的flex线对齐。

# 二、Flex-item

Flex 子元素布局

# 1. order

排序字段，默认为 0

order 由小到大从主轴方向排序，-1 0 1 ....

# 2. align-self

重写 flex-container 中的 align-items 中的属性

align-self: flex-start;

# 3. flex-basis

设定 flex-item 在主轴方向上的大小

// 如果 flex-direction 为 row, 这相当于 width: 100px; 反之 相当于 heigth: 100px;
flex-basis: 100px;

# 4. flex-grow

主要在主轴方向上有剩余空间时，才会生效。

flex-grow: x; 如果 flex-item 的 flex-grow 相同时，则flex-item 平分主轴上的空间.
如果 flex-item 的 flex-grow 不相同，则按照 flex-grow 比例分割主轴上的空间

# 5. flex-shrink

主轴方向上，空间不足时生效，沿着主轴方向缩小

flex-shrink: 默认值 1；表示所有 flex-item 在主轴方向上空间不足时，会按照相同比例缩小；
flex-shrink: 0; 这 flex-item 即使主轴方向上控件不足也不会缩小

# 6. flex

flex = flex-grow + flex-shrink + flex-basis

flex: 1 1 auto;
// 相当于
flex-grow: 1;
flex-shrink: 1;
flex-basis: auto;
