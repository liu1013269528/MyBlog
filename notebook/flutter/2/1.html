TextField 聚焦与失焦状态

FocusNode _focusNode = FocusNode();
@override
void initState() {
  _focusNode.addListener(() {
    if (_focusNode.hasFocus) {
      print("输入框聚焦");
    } else {
      print("输入框失焦");
    }
  });
  super.initState();
}



TextField(

Flutter 解决 MissingPluginException(No implementation found for method xxx on channel xxx)

方案一：flutter clean一下，再次 flutter run/build

方案二：将flutter run的进程终止掉重新 flutter run/build

方案三：看看你的FlutterPluginRegistrant产物是否存在，是否更新了

方案四：杀死APP进程重新运行

master 为当前最新的channel；
dev 为当前最新且充分测试后的channel；
beta 为每月Flutter官方调整选出来的最好的dev的channel；
stable 为当前最稳定的channel。
稳定性排序：master < dev < beta < stable 。
